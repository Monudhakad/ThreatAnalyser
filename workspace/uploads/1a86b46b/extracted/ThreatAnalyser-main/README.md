# ThreatAnalyser
AI-powered backend platform for automated vulnerability detection and security reporting


## Features

- Upload project ZIP files
- Detect technologies and dependencies
- Fetch CVE vulnerabilities
- AI-powered security analysis
- Risk scoring
- Generate PDF reports

## Tech Stack

- Java
- Spring Boot
- REST APIs
- OpenAI API
- NVD CVE Database
- Thymeleaf
- Maven

## Architecture

Client → Controller → Service → Engine → Report Generator
