## ISSUE #1: Naming Consistency Audit

### Finding

The project uses **inconsistent naming**:
- **Folder structure**: Root is "ThreatAnalyser.worktrees" (British: 'ser')
- **Maven module**: "threatanalyzer" (American: 'zer') 
- **Git repository**: "ThreatAnalyser" (British: 'ser')

### Recommendation

**Keep "threatanalyzer" (American spelling)** as the standard because:
1. Maven module already uses this
2. More common in international software
3. Easier consistency in code identifiers
